# 💳 ATM Simulation System (CLI - Python)

A simple Command Line Interface (CLI) based ATM Simulation System built using Python. This project simulates basic banking operations like checking balance, depositing, withdrawing money, and viewing transaction history.

---

## 🚀 Features

- 🔐 PIN authentication system
- 💰 Check account balance
- ➕ Deposit money
- ➖ Withdraw money
- 🧾 View transaction history
- ⚠️ Insufficient balance handling

---

## 🛠️ Tech Stack

- Language: Python
- Interface: Command Line Interface (CLI)
- Data Storage: In-memory (variables / lists)

---

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/thelatika-22/atm-simulation.git


2. Navigate to the folder:

cd atm-simulation


3. Run the program:

python main.py

---

## 💡 How It Works

- User logs in using a PIN
- System displays menu options
- User selects operations like deposit, withdraw, or check balance
- Transactions are updated and stored during runtime
